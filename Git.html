Comandos básicos de GIT:

git --version: Este comando nos dirá la versión de GIT que tenemos instalada.

git clone: Permite clonar un repositorio en un nuevo directorio.

git init: Crea un repositorio vacio o lo inicializa.

git add: Permite subir archivos.

git commit: Captura el estado de un proyecto en ese momento.

git pull: Permite obtener los cambios del repositorio.

git push: Permite subir los cambios al repositorio.
